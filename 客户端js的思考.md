# 什么是javascript
+ 脚本语言
+ 运行在浏览器中
+ 一般是用来做客户端页面的交互（interactive）

# javascript的运行环境
+ 是不是运行在浏览器呢？
+ 不够严谨
+ 运行在浏览器内核中的js引擎中

# 浏览器中的javascript可以做什么？
+ 操作DOM（对DOM的增删改查、注册事件等）
+ ajax（异步发送http请求）
+ BOM（页面跳转、历史记录、console、alert）
+ ECMAScript

# 浏览器中的javascript不可以做什么？
+ 文件操作（文件和文件夹的CRUD）
+ 操作操作系统信息
+ 由于运行环境的特殊性（js文件放置在服务器端，但是运行的时候是在客户端运行的），所以不能做文件的操作和操作操作系统信息

# 编程语言的能力取决于什么？
+ 不是语言本身：语言本身只是提供定义变量、定义函数、定义类型、流程控制、循环结构之类的操作
+ 取决于运行该语言的平台（环境）
+ 对于javascript来说，我们常说的js实际是ECMAScript，大部分的能力都是由浏览器的执行引擎决定的
+ BOM和DOM可以说是浏览器开放出来的接口
+ 比如：cordova中提供了js调用摄像头、操作本地文件的API

- java是语言，java运行在java虚拟机中，java虚拟机称作java VM，简称JVM
- c#是语言，c#运行在.NET Framework中（这个平台只支持windows操作系统），c#也可以运行在MONO中（因为有人想把C#运行在linux平台上，所以开发了这个平台）
- PHP是语言，也是平台

# javascript只可以运行在浏览器中吗？
+ 不是
+ 能运行在哪里取决于：这个环境有没有认同特定平台
+ 语言的能力一定取决于执行代码的平台
